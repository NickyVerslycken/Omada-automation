# Python Omada Automation

GUI tool for batch LAN/VLAN management using Omada OpenAPI.

## Run

```bash
python runners/run_gui.py
```

Or run the launcher:

```bash
python start-omada-app.pyw
```

## Development Notes

### Git Configuration

This repository is configured to ignore:
- **`__pycache__/`** and Python bytecode files (`.pyc`, `.pyo`) - These are auto-generated by Python
- **`.vscode/`** and other IDE-specific folders - These contain personal development environment settings

When you clone this repository on a new machine, Python will automatically recreate `__pycache__` folders as needed. You can configure your own IDE settings without affecting other contributors.
